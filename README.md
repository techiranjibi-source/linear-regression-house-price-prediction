 🏠 House Price Prediction using Linear Regression

 📌 Project Overview
This project demonstrates the implementation of "Simple & Multiple Linear Regression" using Scikit-learn to predict house prices.

---

 🎯 Objective
- Understand regression modeling
- Evaluate model using MAE, MSE, R²
- Interpret coefficients

---

 🛠 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

 📊 Dataset
  Housing dataset with features:
  - Area
  - Bedrooms
  - Bathrooms
  Target:
  - Price

---

 ⚙️ Workflow
1. Data Loading & Cleaning
2. Train-Test Split
3. Model Training (Linear Regression)
4. Prediction
